# Castelet — Как пользоваться

Castelet — это веб-приложение для автоматизации работы с Telegram. Оно позволяет собирать сообщения из каналов и чатов, генерировать новый текст с помощью ИИ и отправлять его обратно в Telegram.

## Что нужно для начала

Перед запуском убедитесь, что у вас есть:

- **Telegram API ID и API Hash** — получите на сайте [my.telegram.org](https://my.telegram.org)
- **Ключ OpenAI API** — получите на сайте [platform.openai.com](https://platform.openai.com)

Эти данные нужно записать в файл `.env` в папке проекта.

## Как запустить

1. Откройте терминал и перейдите в папку проекта
2. Установите зависимости командой `npm install` (только при первом запуске)
3. Запустите приложение командой `npm run ui`
4. Откройте в браузере адрес `http://localhost:3333`

## Как работает приложение

Интерфейс состоит из 4 шагов. Переключайтесь между ними с помощью кнопок-вкладок в верхней части экрана.

### Шаг I — Настройка

Здесь вы увидите статус подключения: зелёные галочки означают, что всё в порядке, красные — что нужно добавить недостающие данные в `.env`. Также здесь можно авторизоваться в Telegram — введите номер телефона и код подтверждения.

### Шаг II — Сбор данных

На этом шаге вы собираете сообщения из Telegram. Укажите:

- Имя канала или чата (например, `@mychannel`)
- Количество сообщений для сбора

Можно собирать как сообщения, так и комментарии к постам. Результат сохраняется в файл автоматически.

### Шаг III — Генерация текста

Здесь ИИ создаёт новый текст на основе собранных данных:

1. Выберите файл с данными (из предыдущего шага)
2. Выберите шаблон промпта — он определяет стиль и задачу для ИИ
3. Нажмите кнопку генерации и дождитесь результата

Сгенерированный текст сохраняется в отдельный файл.

### Шаг IV — Отправка

Отправьте сгенерированный текст в Telegram:

1. Выберите файл с текстом
2. Укажите, куда отправить (комментарий к посту или сообщение в чат)
3. Подтвердите отправку

Длинные сообщения разбиваются автоматически. Между отправками есть задержка, чтобы не попасть под ограничения Telegram.

## Журнал событий

В нижней части экрана отображается лог — здесь видно, что происходит в реальном времени: какие сообщения собираются, как идёт генерация и отправка.
